# Email templates for EDU-Pifagor

Папка содержит готовые шаблоны email-писем для платформы «Пифагор».

## Структура
- `shared/` — базовый HTML-шаблон
- `auth/` — регистрация, подтверждение email, восстановление пароля
- `assignments/` — задания и проверки
- `testing/` — тестирование
- `schedule/` — напоминания из расписания
- `feedback/` — письма по обращениям
- `system/` — системные письма и новости

## Рекомендации
- HTML и TXT версии использовать парами
- общие стили вынести в `shared/base_email.html`
- subject письма хранить в коде или в отдельных text-файлах
- все ссылки и даты передавать через контекст Django

## Пример пути в проекте
`backend/templates/emails/...`
