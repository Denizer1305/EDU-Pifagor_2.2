# Static assets

Папка `static/` содержит backend-ассеты проекта «Пифагор».

## Основные разделы
- `brand/logo/` — логотипы по цветовым схемам
- `brand/watermarks/` — ватермарки по цветовым схемам
- `brand/themes/` — конфигурации тем
- `emails/` — ассеты для email-шаблонов
- `documents/` — preview и служебные документные ассеты
- `admin/` — кастомизация Django admin

## Принципы
- SVG — для логотипов, иконок, ватермарок
- WEBP — для фото, баннеров и тяжелых изображений
- `static/` не используется для пользовательских загрузок
