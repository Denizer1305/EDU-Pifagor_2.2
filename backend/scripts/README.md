# Backend scripts

Папка `scripts/` содержит служебные утилиты для backend-проекта «Пифагор».

## Основные группы скриптов
- bootstrap-скрипты для создания структуры;
- seed-скрипты для начальных данных;
- dev-утилиты;
- export/check-утилиты;
- безопасные обертки для инфраструктурных операций.

## Рекомендации
- каждый скрипт должен быть идемпотентным, если это возможно;
- у каждого скрипта должно быть понятное имя;
- не хранить здесь предметную бизнес-логику приложений.
