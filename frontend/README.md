# Frontend — EDU-Pifagor

Frontend-часть платформы **«Пифагор»** реализована на **Vue 3 + Vite** и отвечает за пользовательский интерфейс, маршрутизацию, кабинеты по ролям, формы, визуализацию расписания и аналитики.

## Назначение

Frontend предоставляет:
- публичные страницы платформы;
- авторизацию и личные кабинеты;
- отдельные интерфейсы для администратора, преподавателя, студента и родителя;
- конструкторы курсов, уроков, заданий и расписания;
- интерфейсы обратной связи;
- дашборды и аналитические экраны.

## Основной стек

- Vue 3
- Vite
- Vue Router
- Pinia
- Axios
- SCSS

Дополнительно могут использоваться:
- vee-validate
- yup
- ApexCharts

## Основные разделы интерфейса

### public
- главная страница;
- о платформе;
- преподаватели;
- контакты.

### auth
- вход;
- регистрация;
- потдверждение эл.почты;
- восстановление пароля.

### teacher
- мои курсы;
- уроки;
- материалы;
- задания;
- тесты;
- расписание;
- аналитика;
- должники.

### student
- мои курсы;
- задания;
- дневник;
- расписание;
- уведомления.

### parent
- успеваемость ребёнка;
- посещаемость;
- расписание;
- уведомления.

### admin
- пользователи;
- организации;
- группы;
- предметы;
- контроль данных;
- обращения пользователей.

## Архитектурные принципы

- маршруты разделяются по ролям;
- логика API выносится в `src/api/`;
- повторно используемый предметный UI — в `src/modules/`;
- базовые и общие компоненты — в `src/components/`;
- layouts отвечают за структуру кабинетов;
- Pinia используется для глобального состояния.
