# Frontend — EDU-Pifagor

Frontend-часть платформы «Пифагор» реализуется на Vue 3 + Vite.

## Основные разделы
- pages
- modules
- components
- widgets
- stores
- api
- themes
- styles
- assets

## Принципы
- страницы должны быть тонкими;
- предметная логика выносится в modules;
- повторно используемые UI-элементы — в components;
- API-вызовы централизуются в src/api;
- тема интерфейса управляется через CSS variables и stores/theme.js.
