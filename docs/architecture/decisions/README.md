# Architecture Decisions

В этой папке фиксируются архитектурные решения проекта в формате ADR (Architecture Decision Record).

## Зачем это нужно

ADR помогают:
- не терять контекст принятых решений;
- объяснять, почему архитектура устроена именно так;
- быстро вводить в проект нового разработчика;
- использовать решения в дипломной документации.

## Формат

Каждый файл должен содержать:
- контекст;
- решение;
- последствия.
