# Diagrams

Папка `diagrams/` содержит визуальные материалы проекта.

## Разделы

### `er/`
ER-диаграммы базы данных.

### `use-cases/`
Диаграммы вариантов использования:
- администратор;
- преподаватель;
- студент;
- родитель.

### `deployment/`
Схемы развертывания:
- локальная разработка;
- docker compose;
- production-контур.

## Правило

Текстовые описания должны дублироваться в `docs/architecture/`, а графические материалы храниться здесь.
