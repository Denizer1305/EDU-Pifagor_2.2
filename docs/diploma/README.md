# Diploma Materials

Папка `diploma/` содержит материалы, связанные с дипломным проектированием.

## Что здесь хранить

- план пояснительной записки;
- черновики разделов;
- таблицы сущностей;
- выдержки по архитектуре;
- ссылки на диаграммы;
- словарь терминов проекта.

## Принцип

Все материалы должны быть организованы так, чтобы:
- их можно было быстро использовать в пояснительной записке;
- архитектурные решения не расходились с кодовой базой;
- изменения в проекте можно было легко отразить в дипломе.
